# 🛒 Retail Anomaly Detection & AI Recruitment Suite

An end-to-end AI system combining **retail anomaly detection** and an **AI-powered recruitment analytics platform**.
This project demonstrates real-world applications of machine learning, NLP, and explainable AI in business analytics and hiring automation.

---

## 🚀 Overview

This repository contains two major components:

### 1️⃣ Retail Anomaly Detection

* Detects unusual sales patterns using **Isolation Forest**
* Identifies spikes, drops, and irregular transactions
* Supports batch and simulated real-time data

### 2️⃣ AI Recruitment Suite

* Resume parsing (PDF + text)
* Skill extraction (NLP-based)
* Experience similarity scoring (TF-IDF)
* Unified candidate ranking system
* Interactive dashboards using Streamlit

## ▶️ Usage

Run anomaly detection pipeline:

```bash
python main.py
```

Run Streamlit dashboard:

```bash

```
